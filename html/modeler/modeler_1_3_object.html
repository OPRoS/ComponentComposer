    다음은 object에 대한 프로퍼티를 변경할 수 있는 창이다.


    그림 2.1-2 proprety 창

    • Attributes - 선택된 객체의 어트리뷰트를 변경한다.
    • BackGround Color - 선택된 객체의 색상을 변경한다.
    • Multiplicity - 선택된 객체의 다양성을 변경한다.
    • Name - 선택된 객체의 이름을 변경한다.
    • Operation - 선택된 객체의 오퍼레이션을 변경한다.
    • Package - 선택된 객체의 패키지 구조를 알려준다.
    • Properties - 선택된 객체의 Tag와 Description을 변경한다.
    • Scope - 선택된 객체의 타입을 변경한다. public, protected, private, package 네가지가 있다.
    • Set Object State - object state를 변경한다.
    • Stereotype - 선택된 객체의 스테레오 타입을 변경한다.
    • Type - element classifier을 변경한다.

관계 연결 및 편집

Sequence 다이어그램은 Sequence의 palett에서 객체를 이용하여 작성한다.

  1. Message

    2. 액터와 오브젝트간에 전송되는 메시지를 나타낸다.


    그림 2.1-2 연관 관계 종류

  2. Self Message

    3. 액터나 오브젝트가 스스로에게 전송하는 메시지를 나타낸다.


    그림 2.1-2 연관 관계 종류

  3. 관계 편집

    4. 관계에 마우스를 갖다 대고 우클릭을 하면 다음과 같은 팝업창이 뜬다.


    그림 2.1-2 관계 팝업창

    • Edit Message - 메세지를 편집한다.
    • Delete - 관계를 삭제한다.

    Edit Message버튼을 누르면 다음과 같은 창이 뜬다.


    그림 2.1-2 관계 팝업창

    • Message - 메시지 종류를 선택
    • Signature - 시그네쳐를 나타내는지를 선택
    • Condition - 상태를 편집한다.
    • Constraint - 제약조건을 편집한다.
    • Synch - 동기인지 비동기인지 편집한다.
    • Kind - 신호인지 아닌지를 정한다.
    • Lifecycle - new, delete, none인지를 정한다.

객체 생성 및 편집

Timing 다이어그램은 Timing의 palett에서 객체를 이용하여 작성한다.

  1. 객체 생성

    2. palette에서 생성할 객체를 클릭하고 원하는 다이어그램 위치에 클릭하면 객체가 생성된다.


    그림 2.1-2 객체 생성

  2. 객체 삭제

    3. 객체삭제는 모델삭제와 뷰 삭제 두가지가 있다. 다이어그램에는 여러개의 같은 객체의 뷰가 생성 가능하지만 모델에는 단 하나이 객체만 생성할 수 있다.

    패키지 모델을 삭제할 때는 트리에서 삭제할 패키지에 마우스를 대고 우클릭을 하면 뜨는 팝업창에서 Delete Model을 선택하면 객체를 삭제할 수 있다. 객체 모델을 삭제하면 추가했던 뷰들도 모두 삭제된다.


    그림 2.1-2 모델 지우기

    뷰 지우기는 다이어그램에서 삭제할 객체에 마우스를 대고 우클릭을 하면 뜨는 팝업창에서 delete를 선택하면 다이어그램에서뷰를 삭제할 수 있다.


    그림 2.1-2 뷰 지우기

  3. 객체 이름 변경

    4. 객체의 이름을 바꾸려면 클래스를 선택하고 이름이 보이는 영역을 더블클릭하면 이름을 변경할수 있도록 선택된다. 변경하고자 하는 이름으로 입력하고 엔터를 누르거나, 객체를 마우스로 클릭하면 이름이 변경된다. 객체의 이름은 다른 객체 이름과 중복되어서는 안된다.


    그림 2.1-2

    5. 하단에 진행상태를 숫자 또는 숫자가 안보이도록 변경한다.

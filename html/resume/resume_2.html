운용 환경

컴포넌트 생성 지원 도구는 Eclipse로 구현되었다. 따라서 Sun MicrosystemsTM의 Java 2 Software Development Kit (SDK)가 설치될 수 있는 환경이면 어떠한 플랫폼에서도 운용 가능 하다. 현재 Microsoft Windows XP를 중심으로 개발되고 테스팅 되었으며, 다음과 같은 운용 환경을 추천한다.

  • 하드웨어 : 펜티엄급 중앙처리장치(CPU) 이상
  • 주메모리 : 128 MB 이상
  • 표시장치 : SVGA 이상 (1024x768 이상 권함)
  • 운영체제 : Microsoft Window 95/98/NT/2000TM, UNIX TM 등
  • 소프트웨어 : Java 2 SDK 1.4.2 혹은 Runtime Environment (JRE) 1.5 이상

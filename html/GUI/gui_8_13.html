객체 찾기

찾기 기능은 텍스트를 중심으로 모델이나 다이어그램을 찾는 기능이다.


그림 2.1-2 찾기 팝업

편집 메뉴에서 find버튼을 누르면 다음과 같은 창이 뜨는데 텍스트를 기반으로 모델을 찾을 수 있다.


그림 2.1-2 찾기 창

  • Text to Find - 찾을 텍스트를 입력한다. 기본적으로 객체의 이름을 찾는다.
  • Model Type - 찾을 객체의 타입을 지정할 수 있다. 세분한 찾기가 가능하다.
  • in Model - 도구 좌측의 트리에서 찾는다.
  • in Diagram - 다이어그램에서 찾는다.
  • Case Sensitive - 대소문자를 구별한다.
  • Exact Match - 모르겠음
  • In Description - 설명에서도 찾는다.

콘솔창에 검색 결과에서 모델을 클릭하면 트리에서 모델이 선택된 것을 볼 수 있다.


그림 2.1-2 트리에서 찾은 모델

다이어그램에서도 모델을 찾을 수 있는데 In Diagram을 선택하면 가능하다.


그림 2.1-2 찾기 창

콘솔창에 검색 결과에서 모델을 클릭하면 다이어그램이 열리고 모델이 선택된 것을 볼 수 있다.

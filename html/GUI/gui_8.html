다이어그램 공통 편집 기능

  • 객체 추가 : 다이어그램에 객체를 추가한다.
  • 객체 선택/다중 선택 : 다이어그램 영역을 축소한다.
  • 객체 이동 : 다이어그램에 가로 세로 수치를 표시한다.
  • 객체 크기 변경 : 다이어그램에 그리드가 보이도록 한다.
  • 객체 복사 : 다이어그램에서 객체를 복사한다.
  • 객체 붙여넣기 : 다이어그램에 복사된 객체를 붙여넣는다.
  • 객체 삭제 : 다이어그램에서 객체를 삭제한다.
  • 객체 이름 변경 : 선택된 객체의 이름을 변경한다.
  • 객체 메뉴 보기 : 선택된 객체에 대한 메뉴를 본다.
  • 객체 속성 편집 : 선택된 객체의 속성을 편집한다.
  • 객체 간 관계 설정 : 다이어그램에서 객체간의 관계를 설정한다.
  • 객체 드래그/드랍 : 트리에있는 모델을 다이어그램에 적용한다.
  • 객체 찾기 : 텍스트를 기반으로 모델을 찾는다.
